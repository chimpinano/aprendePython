Tipos de datos



Variables

Hemos aprendido como imprimir una línea te texto, como obtener el resultado de una operación e incluso hacer operaciones lógicas, pero ¿Qué pasa si queremos almacenar ese resultado para volverlo a utilizar? Para ésto existen las variables.

Una variable en programación es bastante similar a una variable en álgebra y en Python se declaran de la misma manera.

Hablando normalmente:

Me llamo Pablo, tengo 17 años y soy estudiante

En Python mis datos serían puestos de ésta manera: 

nombre = "Pablo"
edad = 17
estudiante = True

Como ves, la forma de declarar una variable en Python es bastante sencilla, basta con poner el nombre de la variable (SIN COMILLAS) seguido del signo igual y el valor que tendrá dicha variable.


Tipos de datos

Con el ejemplo anterior notamos que el valor de cada variable está escrito de manera diferente, el nombre está entrecomillado mientras que el valor de la edad y estudiante no, ésto se debe a que son diferentes tipos de datos, el primero es texto, el segundo es un número y el último es un valor de verdadero o falso.

En otros lenguajes se tendría que especificar el tipo de dato que alamacenará esa variable pero en Python no, en Python basta con el nombre de la variable y su valor. Ésto se debe a que Python es un lenguaje de tipado dinámico, es decir, podemos ahorrarnos escribir el tipo.

Texto

En programación conocido como Cadenas o Strings. Python tiene diferentes maneras de alamacenar texto, ya sea de tipo bytearray o unicode (en Python 3 todas las cadenas son unicode) pero en ésta ocación solo nos enfocaremos a las cadenas de texto normales: 

texto = "Hola, soy un texto"

Python cuenta con la función type() que nos permite saber el tipo de dato de algo, probemos con lo anterior: 

>>>texto = "Hola, soy un texto"
>>>type(texto)
type 'str'

El resultado arrojado nos dice que es del tipo str osea un String. Podemos también sumar las cadenas: 

>>>saludo = "Hola"
>>>nombre = "Pablo"
>>>texto = saludo + ", mi nombre es " + nombre
>>>texto
'Hola mi nombre es Pablo'

Python posee unas funciones muy cool entre las que están:
Hacer mayúscula la primera letra de un String: 

>>>letra = "a"
>>>letra.capitalize()
'A'

Hacer mayúscula o minúscula todo un texto: 

>>>texto = 'EsTe Es Un TeXtO cOn MiNuScUlAs Y mAyUsCuLaS'
>>>texto.upper()
'ESTE ES UN TEXTO CON MINUSCULAS Y MAYUSCULAS'
>>>texto.lower()
'este es un texto con minusculas y mayusculas'

Contar el número de veces que un caracter se repite: 

>>>texto = 'Hola me llamo Pablo'
>>>texto.count('H')
1
>>>texto.count('o')
3

Remplazar un caracter por otro: 

>>>texto = 'Hola me llamo Pablo'
>>>texto.replace('o','#')
'H#la me llam# Pabl#'

Todas las cosas que puedes hacer con un String son geniales, pero si deseas conocer más te recomiendo leer la documentación.

Números

En Python podemos resumir cuatro tipos de números, enteros, largos, flotantes y complejos pero en ésta ocasión solo nos enfocaremos en los enteros (int) y floatentes (float).
Un número entero no contiene decimales y al realizar operaciones entre dos enteros sólo podrás obtener otro entero. En Python se declaran así: 

>>>entero = 17
>>>type(entero)
type 'int'

Los flotantes son también llamados números de coma flotante, éstos siempre son representados con un punto decimal: 

>>>flotante = 3.141592
>>>type(flotante)
type 'float'

Dentro de las funciones que poseen los números encontramos:
Obtener el valor absoluto de un número: 

>>>numero = -3
>>>abs(numero)
3

Convertir un entero a flotante: 

>>>numero = 10
>>>float(numero)
10.0

Convertir un flotante a entero: 

>>>numero = 3.141592
>>>int(numero)
3

Convertir un entero a binario (en un string): 

>>>numero = 2014
>>>bin(numero)
'0b11111011110'

Convertir un entero a hexadecimal (en un string): 

>>>numero = 31251
>>>hex(numero)
'0x7a13'

Convertir un entero a octal (en un string): 

>>>numero = 2343
>>>oct(numero)
'04447'

Conocer el cociente y resto respectivamente de dos números, dividendo y divisor respectivamente: 

>>>divmod(4,5)
(0, 4)

Conocer el mayor de dos números: 

>>>max(5,6)
6

Conocer el menor de dos números: 

>>>min(5,6)
5

Exponenciar un número, base y exponente respectivamente: 

>>>pow(3,4)
81

Redondear un flotante, número por redondear y digitos despues del punto decimal respectivamente: 

>>>round(3.141592,2)
3.14

Si deseas saber más acerca de las funciones existentes te recomiendo checar la documentación.

Booleanos

Los booleanos son el tipo de dato más sencillo que puede existir, éstos solo pueden tener dos valores, True (verdadero) y False (falso) y la forma de declararse es así: 

verdadero = True
falso = False

Es muy importante que cuando declaremos un valor booleano los valores no estén entrecomillados y se escriban siempre con la primera letra en mayúscula y las demás en minúscula, de otra manera Python no sabrá que es un booleano. 

>>>type(True)
type 'bool'

La función más común para los valores booleanos es Not, la cuál niega el valor actual y lo convierte al valor opuesto: 

>>>booleano = True
>>>not booleano
False
