Primeros Pasos



Área de trabajo

Antes de comenzar necistaremos definir sobre que probaremos nuestros códigos, existen dos formas de correr código Python, una es sobre el intérprete o la otra es ejecutando un archivo con extensión .py que contendrá todo nuestro código.

A medida que crece nuestro código y la complejidad del mismo aumenta, es preferible que todo lo almacenemos en un archivo.py, ya que cuando corremos código sobre el intérprete y después lo cerramos todo el código se habrá perdido. Como consejo recomiendo el intérprete para probar cosas simples que no nos importa perder.

Corriendo sobre el intérprete.

Para abrir el intérprete es necesario que usemos la consola y tecleemos el comando siguiente con el que estarémos listos para escribir nuestro código dentro de la consola: 

$ python

Ejecutando un archivo.py

Con cualquier editor de texto, puede ser Sublime Text (el que yo uso), TextMate, Brackets, Notepad++, vim o nano (si son "jarcor") o cualquier otro escribirás tu código Python y guardarás tu archivo con el siguiente formato nombre_del_archivo.py, lo cual lo deja listo para ser usado en cualquier momento que desees.

Para usar tu archivo.py una vez creado, tendrás que abrir la consola, ubicarte en la carpeta que contiene tu archivo y ejecutar el siguiente comando que correrá todo el código que adentro esté escrito: 

$ python archivo.py

Imprimiendo Líneas

Lo primero que generalmente hacemos cuando estamos aprendiendo un nuevo lenguaje de programación es hacer el famoso "Hola mundo!". Imprimir líneas en Python es requiere de dos pasos, la sentencia print y el texto que queremos imprimir entre comillas, al menos por el momento. 

print "Hola mundo!"

Lo cual imprimirá: 

Hola mundo!


El texto que está entrecomillado es conocido en la programación como cadenas o strings. Si deseas imprimir dos diferentes cadenas a la vez (conocido también como concatenación de cadenas) basta con hacer algo similar a lo siguiente: 

print "Hola mundo!", "¿Cómo estás?"

Lo cual imprimirá: 

Hola mundo! ¿Cómo estás?


Operaciones aritméticas

En cualquier lenguaje de programación tenemos la posibilidad de hacer operaciones matemáticas de todo tipo y Python no es la excepción. Las operaciones básicas que Python maneja son: sumas, restas, multiplicaciones, divisiones, conocer el cociente en una división, conocer el módulo o resto en una división y hacer exponenciación de un número.

Sumas y restas

Queremos ejecutar la suma:

20 + 6 = 26

En Python sería algo así: 

>>> 20 + 6
26

Queremos ejecutar la resta:

20 - 6 = 14

En Python sería algo así: 

>>> 20 - 6
14


Multiplicaciones y divisiones


Queremos ejecutar la multiplicación:

20 X 6 = 120

En Python sería algo así: 

>>> 20 * 6
120

Queremos ejecutar la división:

20 / 6 = 3.33

Para ésta operación veremos algo interesante 

>>> 20 / 6
3         

>>> 20.0 / 6.0
3.33



Cociente y resto


Para los que no recuerdan, el cociente de una división no siempre es el resultado final. En una división la parte entera del resultado también es nombrado cociente.

20 / 6 = 3.33       Cociente = 3

En Python lo calculamos así: 

>>> 20 // 6
3

El resto o mejor conocido como módulo en programación, se usa generalmente para saber si un número es múltiplo de otro. En matemáticas el módulo sería el resultado de la resta del dividendo y la multiplicación del cociente por el divisor o más simple el residuo.

20 % 6 = 3.33       Resto ó módulo = 2

En Python lo calculamos así: 

>>> 20 %  6
2
