Primeros Pasos



Área de trabajo

Antes de comenzar necistaremos definir sobre que porbaremos nuestros códigos, existen dos formas de correr código Python, una es sobre el intérprete o la otra es ejecutando un archivo con extensión .py que contendrá todo nuestro código.

A medida que crece nuestro código y la complejidad del mismo aumenta, es preferible que todo lo almacenemos en un archivo.py ya que cuando corremos código sobre el intérprete y después lo cerramos todo el código se habrá perdido. Como sonsejo recomiendo el interprete para probar cosas simples que no nos importa perder.

Corriendo sobre el intérprete.

Para abrir el interprete es necesario que usemos la consola y tecleemos el comando siguiente y estarémos listos para escribir nuestro código dentro de la consola: 

$ python

Ejecutando un archivo.py

Con cualquier editor de texto, puede ser Sublime Text (El que yo uso), TextMate, Brackets, Notepad++, vim o nano (Si son "jarcor") o cualquier otro escribirás tu código Python y guardarás tu archivo con el siguiente formato nombre_del_archivo.py, lo cuál lo deja listo para ser usado en cualquiermomento que desees.

Para usar tu archivo.py, una vez creado por su puesto, tendrás que abrir la consola, ubicarte en la carpeta que contiene tu archivo y ejecutar el siguiente comando que ejecutará todo el código que adentro esté escrito: 

$ python archivo.py
